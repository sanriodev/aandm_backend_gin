The classic `aandm-API` rewritten in go for simplicity and blazingly fast performance!

### Api basepath

`/api/v1` on port `3000`

### to run locally

```zsh
$ go run main.go
```

### to build into executable

```zsh
go build
```

## OpenAPI Documentation

### generate Swagger Documentation

```zsh
swag init
```

### format swag comments

```zsh
swag fmt
```
